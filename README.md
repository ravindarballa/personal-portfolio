## Microfrontend Communication
- **Module Federation** is used for sharing modules/components across Angular and React.
- Angular shell dynamically loads React microfrontend components.
- Backend APIs provide data via REST endpoints.

## Backend Services
- **NestJS/Express** backend
- MongoDB or SQL database for persistent storage
- REST APIs for project data, blog posts, contact messages
- JWT authentication for admin features

## CI/CD Pipeline
- GitHub Actions for automated testing and deployment
- Nx affected commands to only build changed projects

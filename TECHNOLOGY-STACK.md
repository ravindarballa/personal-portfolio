# Technology Stack

## Frontend
| Technology | Usage |
|------------|-------|
| Angular    | Shell and top navigation |
| React      | Portfolio pages (About Me, Projects, Contact) |
| Next.js    | Blog section |
| TailwindCSS / SCSS | Styling and responsive UI |
| Nx         | Monorepo management and build orchestration |
| Module Federation | Microfrontend integration |

## Backend
| Technology | Usage |
|------------|-------|
| Node.js / NestJS | REST API server |
| Express.js | HTTP request handling |
| MongoDB / SQL | Database storage |
| JWT        | Authentication |
| Docker     | Containerization |

## DevOps
| Technology | Usage |
|------------|-------|
| GitHub Actions | CI/CD automation |
| Nx CLI      | Build & serve microfrontends |
| Docker     | Deployment & container management |
| Kubernetes | Optional scaling |

## Tools
- VS Code / WebStorm
- Postman / Insomnia for API testing
- Nx Console plugin
